## Projects Section: Cerno

**Cerno - AI chess coach for Lichess players**

Cerno is a portfolio MVP that explores how chess engines, retrieval systems, and LLM-based reasoning can work together to turn chess games into personalized training recommendations.

The backend can analyze Lichess games or manually provided PGN games, run Stockfish analysis, store user history, build weakness profiles, retrieve relevant chess theory from ChromaDB, and generate training recommendations through an LLM.

**Technologies:** Python, FastAPI, PostgreSQL, ChromaDB, Stockfish, python-chess, OpenAI API, Docker, Docker Compose, Next.js, React.

## Projects Section: Final Degree Project

**Meta-learning for the optimal selection of univariate feature selection filters in classification tasks**

My final degree project explored whether meta-learning could help recommend the most suitable univariate feature selection filter based on dataset characteristics.

The work involved OpenML datasets, meta-features, Chi-squared, ANOVA F-test, Mutual Information, cross-validation, Random Forest, Gradient Boosting, Soft Voting Ensembles, and analysis of model performance across different dataset groups.

The results were moderate, but the project was valuable as a machine learning experimentation exercise: it reinforced the importance of validation design, honest result interpretation, and understanding the limits of generalization.
