## Featured Project

### Cerno

Cerno is an AI chess coach for Lichess players. It analyzes games, detects recurring weaknesses, retrieves relevant chess theory, and generates personalized training recommendations.

The project explores how chess engines, retrieval systems, and LLM-based reasoning can work together to turn game analysis into practical improvement plans.

Current stack:

- Python, FastAPI
- PostgreSQL
- ChromaDB
- Stockfish and python-chess
- OpenAI API
- Docker and Docker Compose
- Next.js / React frontend

What it demonstrates:

- Backend API design
- Integration with external chess and AI tools
- RAG-based knowledge retrieval
- Engine-assisted game analysis
- Persistent user history and weakness profiling
- MVP product thinking around an AI-assisted learning tool
